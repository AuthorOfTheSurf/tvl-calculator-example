# Solend Total Value Locked Calculator Demo

Pure javascript utility to compute total deposits and borrows from on-chain data.

Useful as a demo for reading on-chain Solend data.

## Setup

    yarn

## Run Demo

    node index.js

Example output:

    Total Deposits:

    ETH 114515764
